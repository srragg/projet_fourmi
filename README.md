# Documentation 

## Projet : La Fourmi de Langton 

L'objectif de ce projet est de modéliser le comportement de la Fourmi de Langton, un automate cellulaire, à l’aide du langage Python et de la bibliothèque Tkinter pour la visualisation sur une interface graphique.

## Objectif 

Le but est de simuler le comportement d'une fourmi qui se déplace sur une grille composée de cases noires et blanches, en suivant des règles simples :
 
 1) Si la fourmi est sur une case noire : elle tourne de 90° vers la gauche, change la couleur de la case en blanc et avance d’une case.
 2) Si la fourmi est sur une case blanche, elle tourne de 90° vers la droite, change la couleur de la case en noir et avance d’une case.
